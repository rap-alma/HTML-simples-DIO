Estruturas básicas do HTML

Voltar ao inicio

Antes de começarmos a falar da estrutura em si, é necessário que você crie uma pasta no seu computador dedicada ao projeto que irá iniciar.

Feito isso, você irá criar um novo arquivo dentro desta pasta chamado "index.html". Você pode colocar outro nome, mas este nome se tornou padrão entre os desenvolvedores para indicar que esta é a página principal do site.

Sobre a estrutura do HTML, ela é reprensentada por 3 pontos. Ou melhor, por três "tags" principais. Uma "Tag" nada mais é que um elemento representado por um sinal de menor "<", o nome da tag e um sinal de maior ">" .

A primeira tag principal é a tag <html></html>. Ela é a tag que diz ao navegador que tudo que há dentro dela é um html. Quando a tag apresenta uma barra "/", significa que é uma tag de fechamento.

A segunta tag principal é a <head></head>. Significa "Cabeçalho" em português. É responsável por pre-carregar a página antes de exibir pro usuário.

A terceira é a <body></body>. Tudo que de fato vai aparecer para o usuário quando navegar pela página.

Os três juntos são representados assim:

.<html>

..<head>

....<body>

....</body>

..</head>

.</html>

Falando sobre tags

Voltar ao inicio

Tags são elementos básicos do HTML que definem a estrutura e o conteúdo de uma página web. Elas são representadas por palavras-chave entre colchetes angulares como <tag>, e podem ter atributos que modificam seu comportamento.

"As tags permitem que os navegadores interpretem e exibam o conteúdo corretamente."

Explicação das tags

Abaixo está uma breve explicação das principais tags:

  • <h1> até <h6>: Definem títulos, com <h1> sendo o mais importante e <h6> o menos importante.
  • <p>: Define um parágrafo de texto.
  • <mark>: Destaca o texto com uma cor de fundo, geralmente amarelo.
  • <small>: Exibe o texto em um tamanho menor que o padrão.
  • <i>: Aplica formatação itálica ao texto.
  • <u>: Sublinhar o texto.
  • <strong>: Dá ênfase forte ao texto, exibindo-o em negrito.
  • <ol>: Cria uma lista ordenada (numerada).
  • <ul>: Cria uma lista não ordenada (com marcadores).
  • <li>: Define um item de lista dentro de <ol> ou <ul>.
  • <a>: Cria um link de hipertexto.
  • <hr>: Insere uma linha horizontal.
  • <sub>: Exibe o texto como subscrito.
  • <sup>: Exibe o texto como sobrescrito.
  • <blockquote>: Define uma citação em bloco.
  • <font>: Define a fonte, cor e tamanho do texto (obsoleto no HTML5).
  • <del>: Marca o texto como excluído, geralmente exibido com um traço no meio.
  • <abbr>: Indica uma abreviação, podendo exibir o significado completo ao passar o mouse.

Atributos básicos

Voltar ao inicio

Atributos nada mais são que propriedades que as tags podem ter. Existem atributos que são mais genericos e podem ser atribuidos a várias tags. Também existem atributos que são mais específicos que apenas uma ou poucas tags podem ter.

Atributos globais

O (id=" " ) é um atributo identificador. Quando se quer aplicar um efeito em java script, utilizamos o identificador para indicar em qual elemento será aplicado.

O (style=" " ) é um atributo que serve para estilizar o elemento que está entre a tag. Em outras palavras, serve para colocar comandos CSS diretamente na tag.

Atributos específicos

O (type=" " ) Um atributo específico da tag <input> que serve para indicar qual estilo de entrada de dados do input. O "input" exibe um campo de entrada de dados.

Links

Voltar ao inicio

Para colocarmos um link na página usamos a tag <a href="algumlink.com">. O atributo href é onde você vai direcionar pra onde o usuário vai após o clique, colocando o link desejano no lugar do "algumlink.com".

Também existe um atributo chamado (target=" " ) que serve para escolher o comportamento do link após o clique.

"Self": O link exibe comportamento padrão e carrega na mesma página

"Blank": O link é aberto em uma nova guia.
